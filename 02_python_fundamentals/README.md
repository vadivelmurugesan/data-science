### Data Types

* int - either negative or positive numbers

* float - Numbers with decimals

* str - String of Characters

* bool - Boolean (True / False)

### type(expression) Function

type(1)    -> int
type(1.1)  -> float
type('h')  -> str
type(True) -> bool
